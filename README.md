💬 Real-Time Chat Application

A full-stack real-time chat application built with modern web technologies. This project enables users to communicate instantly with live updates and online user tracking.

✨ Features
🔐 User Authentication (JWT-based)
💬 Real-time messaging with Socket.IO
🟢 Online users indicator
🧑‍💼 User profile update
⚡ Fast and responsive UI
🌐 Fully deployed (Frontend + Backend)


🛠️ Tech Stack
Frontend
React (Vite)
Tailwind CSS
Axios
Socket.IO Client
Backend
Node.js
Express.js
MongoDB (Mongoose)
JWT Authentication
Socket.IO
